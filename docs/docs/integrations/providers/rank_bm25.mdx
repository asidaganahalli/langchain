# rank_bm25

>[rank_bm25](https://github.com/dorianbrown/rank_bm25) is an open-source collection of algorithms 
> for querying a set of documents and returning the ones most relevant to the query. 
> The most common use case for these algorithms is to create search engines.
>
>So far the algorithms that have been implemented are:
>
>*  Okapi BM25
>*  BM25L
>*  BM25+
>*  BM25-Adpt
>*  BM25T


## Installation and Setup

First, you need to install `rank_bm25` python package.

```bash
pip install rank_bm25
```
